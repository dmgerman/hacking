                     Lynx source distribution and potpourri

   Lynx is the text web browser. This is the top level page for the Lynx
   software distribution site hosted by the Internet Software Consortium.

   The [1]current development sources have the latest version of Lynx
   available (development towards 2.8.8). The [2]main help page for
   lynx-current is online; the current [3]User Guide is part of the online
   documentation.

   The most recent stable release is [4]lynx2-8-7. The [5]main help page
   is online, as well as the [6]User Guide.

   Other resources include:
     * [7]ftp and http mirrors
     * [8]Mailing list archives
     * [9]pgp/gpg signatures
     __________________________________________________________________

   Viewable with any browser; [10]valid HTML.
